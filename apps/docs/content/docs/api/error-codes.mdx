Orbyt utilizes standard HTTP status codes to communicate execution 
failures. Every error response is accompanied by a descriptive message 
identifying the source of the issue.

## Error Triage

When a request fails, the engine attempts to categorize the error 
to determine if a retry or a fallback is appropriate. If an error 
is persistent across all providers, the gateway returns the final 
consolidated status code to the client.
